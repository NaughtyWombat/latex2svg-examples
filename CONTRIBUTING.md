# Contributing

Contributions are welcome when they add focused, reusable LaTeX snippets.

## Snippet Guidelines

- Keep each example focused on one rendered asset.
- Avoid full papers, document classes, bibliography setup, and unrelated prose.
- Put package requirements in the pull request description when the snippet needs them.
- Prefer examples that work with a clear preset: equation, TikZ, table, algorithm, or text.
- Use descriptive filenames such as `gaussian-integral.tex` or `flowchart.tex`.

## Preview Guidelines

If you add a rendered preview, place it in `previews/` with a matching name.

Example:

- source: `tikz/flowchart.tex`
- preview: `previews/flowchart.svg`
