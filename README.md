# LaTeX2SVG Examples

Example LaTeX snippets for rendering equations, TikZ diagrams, tables, algorithms, and technical text as SVG, PNG, or PDF.

Use them with [LaTeX2SVG](https://latex2svg.com), an online LaTeX to SVG converter for technical graphics.

## Usage

Copy any snippet into [LaTeX2SVG](https://latex2svg.com/editor), choose the matching preset, render, and export SVG, PNG, or PDF.

Use the equation preset for files in `equations/`, the TikZ preset for `tikz/`, the table preset for `tables/`, the algorithm preset for `algorithms/`, and the text preset for `text/`.
